# CDN links (jsdelivr)

### Full Build

##### BaseEx
###### esm
```js
import BaseEx from "https://cdn.jsdelivr.net/npm/base-ex@latest/dist/base-ex.esm.min.js";
```
###### iife
```html
<script src="https://cdn.jsdelivr.net/npm/base-ex@latest/dist/base-ex.iife.min.js"></script>
```
